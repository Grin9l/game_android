# Как добавить изображение Ксюши в игру

## Инструкция

### Шаг 1: Подготовьте изображение

1. Найдите изображение девочки Ксюши
2. Сохраните его в формате PNG или JPG
3. Рекомендуемый размер: 200x200 пикселей или больше (квадратное изображение)
4. Назовите файл: `ksyusha.png` или `ksyusha.jpg`

### Шаг 2: Поместите изображение в проект

**Вариант A: Через файловую систему**

1. Скопируйте файл изображения
2. Вставьте его в папку: `app/src/main/res/drawable/`
3. Убедитесь, что файл называется `ksyusha.png` (или `ksyusha.jpg`)

**Вариант B: Через Android Studio**

1. Откройте проект в Android Studio
2. Перейдите в `app/src/main/res/drawable/`
3. Правой кнопкой мыши → New → Image Asset
4. Или просто перетащите файл в папку `drawable`

### Шаг 3: Проверьте имя файла

- Файл должен называться **точно** `ksyusha.png` или `ksyusha.jpg`
- Имя файла должно быть **строчными буквами** (без заглавных)
- Не используйте пробелы или специальные символы в имени

### Шаг 4: Пересоберите APK

После добавления изображения:
1. Запустите сборку APK через GitHub Actions
2. Или соберите локально: `gradlew.bat assembleDebug`
3. Установите новый APK на телефон

## Структура папок

```
app/
  src/
    main/
      res/
        drawable/
          ksyusha.png  ← Поместите изображение сюда
```

## Требования к изображению

- **Формат:** PNG (рекомендуется) или JPG
- **Размер:** Любой, но рекомендуется 200x200 - 400x400 пикселей
- **Прозрачность:** PNG с прозрачностью поддерживается
- **Имя файла:** `ksyusha` (без расширения в коде, Android сам найдет .png или .jpg)

## Если изображение не отображается

1. Проверьте, что файл находится в правильной папке: `app/src/main/res/drawable/`
2. Проверьте имя файла - должно быть `ksyusha.png` или `ksyusha.jpg`
3. Убедитесь, что файл добавлен в Git (если используете)
4. Пересоберите APK после добавления изображения
5. Проверьте логи - должно быть сообщение "Player image loaded successfully"

## Альтернатива: Использование разных разрешений

Для лучшего качества на разных экранах можно создать несколько версий:

- `drawable-mdpi/ksyusha.png` - для средних экранов
- `drawable-hdpi/ksyusha.png` - для высоких разрешений
- `drawable-xhdpi/ksyusha.png` - для очень высоких разрешений
- `drawable-xxhdpi/ksyusha.png` - для экстра высоких разрешений

Но достаточно и одного файла в `drawable/` - Android автоматически масштабирует его.
